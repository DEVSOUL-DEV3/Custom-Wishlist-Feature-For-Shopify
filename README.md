# Custom Ecommerce Wishlist

A custom wishlist feature built for eCommerce websites that allows users to add or remove products from their wishlist with real-time notifications.

## 🚀 Features

- Add products to wishlist
- Remove products from wishlist
- Toast notifications for user actions
- Custom wishlist icons
- Lightweight frontend implementation
- No external plugins required

## 🛠 Technologies Used

- JavaScript
- HTML
- CSS
- liquid


## ⚙️ Setup Instructions

1. Download or clone this repository.
2. Open all the project files.
3. Follow the instructions provided at the **top of each file** to correctly integrate the wishlist functionality.
4. Make sure to add the **wishlist icon PNG file** inside the **assets folder**.
5. After completing the instructions, open the project in your browser to test the wishlist feature.

## 📁 Assets Required

- `wishlist.png` → Place this file inside the **assets** folder.
